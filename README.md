# CareerHub - Job Application Portal

This is a **Job Application Portal** developed as part of my internship at **Internee.pk**.  
The portal allows candidates to apply for jobs through a **multi-step form**, providing their **personal info, education, experience, skills, portfolio links, and resume upload**.

## Features

- Multi-step application process:
  1. Personal Information
  2. Education
  3. Experience & Skills
  4. Review & Submit
- **Mandatory Resume Upload** to ensure recruiters get complete candidate profiles
- Optional Portfolio / LinkedIn field
- Skills and experience input for better candidate profiling
- **Live summary** review before submission
- Responsive and modern UI design

## Technologies Used

- HTML, CSS, JavaScript (Frontend)
- localStorage for data persistence across steps
- (Optional for future) Backend integration for storing applications
